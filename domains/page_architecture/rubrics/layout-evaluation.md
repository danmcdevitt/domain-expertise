# Layout Evaluation Rubric

## Purpose
Evaluate visual layout and design for conversion effectiveness.

## Evaluation Criteria

### 1. Attention Ratio (Weight: 30%)
- Is there one primary focus per section?
- Are distracting navigation elements removed?
- Does everything support the single conversion goal?
- Is the eye drawn to CTAs naturally?

**Attention Ratio Target:** 1:1 (one goal, one link type)

**Red Flags:**
- Multiple navigation options
- Footer links to other pages
- Sidebar with competing content
- Multiple different CTAs

### 2. Visual Hierarchy (Weight: 30%)
- Do larger elements carry more important information?
- Is headline most prominent text?
- Does CTA stand out from surroundings?
- Is there clear distinction between sections?

**Red Flags:**
- Flat design where everything is same prominence
- CTA doesn't stand out
- Subtext more prominent than headlines
- Confusing section boundaries

### 3. Whitespace Usage (Weight: 20%)
- Is there adequate padding around elements?
- Does whitespace create focus?
- Is the page scannable without feeling cramped?
- Is text readable with proper line spacing?

**Red Flags:**
- Cramped, cluttered layout
- Text touching edges
- No breathing room between sections
- Overwhelming amount of content per screen

### 4. Mobile Optimization (Weight: 20%)
- Does content stack properly on mobile?
- Is CTA visible above fold on mobile?
- Are touch targets adequately sized (44px min)?
- Does page load quickly on mobile connections?

**Red Flags:**
- Horizontal scroll on mobile
- Tiny tap targets
- CTA hidden below fold
- Desktop-only formatting

## Layout Checklist

[ ] No top navigation (or minimal, collapsed)
[ ] No sidebar distractions
[ ] Minimal footer links (or none)
[ ] Consistent grid alignment
[ ] Clear section separations
[ ] Generous whitespace throughout
[ ] Single column layout (or clear visual path)
[ ] Mobile-first responsive design

## Visual Contrast Requirements

| Element | Contrast Ratio | Purpose |
|---------|---------------|---------|
| CTA Button | High (stand out) | Draw attention to action |
| Headlines | High | Quick scanning |
| Body Text | Medium | Comfortable reading |
| Background Sections | Low variation | Visual separation |

## Scoring Guide

| Score | Description |
|-------|-------------|
| 9-10  | Clean, focused, mobile-optimized, perfect hierarchy |
| 7-8   | Good layout with minor mobile or hierarchy issues |
| 5-6   | Adequate but cluttered or weak hierarchy |
| 3-4   | Confusing layout, poor mobile, distracting elements |
| 1-2   | Chaotic, no hierarchy, broken on mobile |
